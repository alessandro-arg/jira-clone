# <img src="https://github.com/alessandro-arg/coordina/blob/main/public/logo.svg" alt="Coordina logo" width="200" color="white" /> - Project & Task Management Platform

Coordina is a modern, visually polished **project and task management platform** inspired by tools like Jira, Trello, and Linear.  
Built with **Next.js 16, React 19, Tailwind CSS 4, Hono, Appwrite, and React Query**, it offers an intuitive Kanban workflow, analytics dashboards, and flexible project organization.

> A sleek and powerful way to plan, track, and manage tasks - individually or collaboratively.
